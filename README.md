# プロダクト発想AI

## 概要・説明

**プロダクト発想AI**は、最新のニュースやSNSのトレンドをAIが分析し、世の中の「不満・不安・不便（3F）」を解決するための **1日で実装可能なMVP（Minimum Viable Product）アイデア** を自動提案するWebアプリケーションです。

「自律型プロダクトマネージャー」として、トレンドの抽出からターゲット選定、主要機能の定義、UI構成、推奨技術スタック、さらには「計算・翻訳・判定・マッチング」のフレームワークを用いた選定理由の論理的分析までを一貫して行います。
フロントエンドのみ（シングルファイル）で動作し、APIキーをローカルに保存して利用するセキュアかつ軽量な設計となっています。

## 主な機能一覧

- 💡 **AIによるMVP要件の自動生成**
  - 世の中の3F（不満・不安・不便）から3つのトピックを抽出。
  - 最もインパクトがあり実現可能なベストアイデアを1つ選定し、アプリ名、コンセプト、ターゲット、主要機能、画面構成、技術スタックを詳細に定義。
- 📰 **最新ニューストレンドの取得 (オプション)**
  - GNews APIと連携し、現実の最新ニュースに基づいた高精度なアイデア出しが可能。
- 🧠 **フレームワーク分析**
  - 選定されたアイデアを「計算・翻訳・判定・マッチング」の4つの観点から論理的に評価。
- ⚡ **快適なユーザー体験 (UX/UI)**
  - Web Workerを活用した非同期処理により、AIの長文生成中もUIがフリーズしません。
  - Wake Lock APIを利用し、生成中のスマートフォンの画面スリープを防止。
  - レジューム機能により、生成中に誤ってアプリを閉じても次回起動時に復帰可能。
- 💾 **セキュアなローカルデータ管理**
  - 履歴やAPIキーはすべてブラウザのIndexedDBに保存され、外部サーバーには送信されません。
  - 生成履歴の閲覧機能と、5世代までの自動バックアップ＆復元機能。
  - JSON形式でのデータのエクスポート / インポート機能。
- 📱 **PWA / モバイルファースト設計**
  - Tailwind CSSによるモダンでクリーンなグラスモーフィズムUI。
  - スマートフォンでの利用に最適化されたレイアウトとPWA対応（ホーム画面に追加可能）。

## 使用技術・ライブラリ

### フロントエンド
- **HTML5 / CSS3 / Vanilla JavaScript** (フレームワーク非依存の1ファイル構成)
- **Tailwind CSS** (CDNによるスタイリング)
- **Font Awesome** (アイコン)
- **Google Fonts** (Inter, Noto Sans JP)

### Web API & ブラウザ機能
- **Google Gemini API** (`gemini-2.5-flash`) - AIによるアイデア・要件生成
- **GNews API** - 最新ニュースの取得
- **Web Worker** - バックグラウンドでのAPI通信・JSONパース処理
- **IndexedDB** - 設定データ、生成履歴、バックアップの永続化
- **Wake Lock API** - 処理中の画面消灯防止
- **File API (FileReader)** - 履歴データのエクスポート/インポート

## セットアップ・使い方

本アプリはサーバー構築不要で、ブラウザ上で直接動作します。

### 1. 準備
1. 本プロジェクトのHTMLファイルをダウンロード、またはローカルに作成（例: `index.html`）して保存します。
2. ブラウザ（Chrome, Edge, Safariなど）で `index.html` を開きます。

### 2. 初期設定（APIキーの登録）
1. 画面右上の **「歯車アイコン（設定）」** をクリックします。
2. **Gemini API Key (必須)**: [Google AI Studio](https://aistudio.google.com/) にて取得したAPIキーを入力します。
3. **GNews API Key (オプション)**: [GNews.io](https://gnews.io/) で取得したAPIキーを入力します（入力すると実際の最新ニュースを反映した精度の高い出力になります）。
4. 「保存して閉じる」をクリックします。※キーはブラウザのローカルデータベースのみに保存されます。

### 3. アイデアを発想する
1. トップ画面の **「発想を開始する」** ボタンをクリックします。
2. AIがトレンドを収集・分析し、バックグラウンドで処理を行います（数秒〜十数秒かかります）。
3. 処理が完了すると、抽出された3つのトピックと、ベストアイデアの詳細（機能、UI、技術スタック、分析結果）が画面に表示されます。

### 4. 履歴の確認とデータ管理
- **履歴の確認**: 左上の **「時計アイコン」** をクリックすると、過去に生成したアイデアの一覧を確認・再表示できます。
- **データ移行・バックアップ**: 設定画面から、これまでの履歴と設定をJSONファイルとして「エクスポート」したり、別端末で「インポート」したりすることができます。万が一に備えた自動バックアップの「復元」も設定画面の下部から可能です。
